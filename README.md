# ccc_simulator

Simple client-side simulator UI for browsing the CCC pages and assets.

Open `ccc_home_page.html` in a browser to view the simulator.

**Contents**
- `ccc_home_page.html`: main entry page
- `Login_files/`, `Sign In_files/`, `University Of Rochester_files/`: supporting CSS, JS and asset folders
